# AI 仓颉开发性能测试集

本仓库为 AI 开发仓颉项目提供了一些测试任务，以对比不同 AI 工具和 Skills 等的开发性能（Token 消耗/质量等）

每个目录中有任务书（`task.md`）、测试代码（`xx_test.cj`）和参考实现，任务书中给出了详细的任务描述和明确的规格/约束/目标，同时按仓颉单元测试框架提供了充足的测试用例，并在任务书中要求 AI 必须使用这套测试代码。任务书和测试代码的严格约束，尽可能收束迭代路径，让 AI 开发相关指标有可比性。

使用时，请选择某个任务目录，其中只保留任务书和测试代码。配置好相关 Skills，在任务目录打开 AI 开发工具，让 AI 执行 `task.md` 中的任务，还可以强调只能使用给定的测试代码、不允许修改测试代码。
